# Contributing

This repository is a **frozen archive** of completed academic and engineering
work. It is preserved as a record rather than maintained as a living tool, so it
does not accept feature contributions.

## Reporting

- **Security or integrity issues:** follow [`SECURITY.md`](SECURITY.md).
- **Factual corrections** (a broken link, a documentation error): open a concise
  issue where issues are enabled, or contact the author below.

Pull requests that change program behaviour will not be merged, so that the
archive keeps reflecting the work as originally completed.

## Citation

If you reference this work, please cite it via the metadata in
[`CITATION.cff`](CITATION.cff).
